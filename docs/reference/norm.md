# Euclidean norm

Generic for computing the Euclidean (L2) norm.

Euclidean norm of a sparse_numeric vector.

## Usage

``` r
norm(x, ...)

# S4 method for class 'sparse_numeric'
norm(x, ...)
```

## Arguments

- x:

  A sparse_numeric object.

- ...:

  Further arguments (unused).

## Value

A numeric scalar giving the norm.
